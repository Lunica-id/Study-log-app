## Study Log App

This is the application that allow users to record study time and contents by date and category. 
学習内容を日付・カテゴリ別に記録できる学習ログアプリ。

### Main Features / 主な機能
- Add learning log / 学習ログの追加
- Display total study time / 学習時間の合計表示
- Save data using localStorage / LocalStorageによるデータ保存

### Technologies used / 使用技術
- HTML
- CSS
- JavaScript

### Planned Features / 今後追加したい機能
- 編集・削除機能
- カレンダー表示
